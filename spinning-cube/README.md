# Spinning Cube

> I started this project to learn more about how computers render 3D objects. This is my first ever serious C programs, if you can call it that. It's also my first time ever trying graphics programming. 
> I think I will carry on by starting with a fresh project that implements some more proper practices.
> I learned A LOT about C and have a basic understanding of how this math works now. I am excited to try to implement more proper systems in a future project. Triangles, cross-product, normals, back face culling...

## 🚀 Features
* **Feature A**: Renders 3D objects with perspective.
* **Feature B**: All 3 elementary matrix rotations work. 

## 🛠️ Tech Stack
* **Language:** [C]
* **Library:** [SDL2]
